제품, 콘텐츠 및 서비스에 대한 손쉬운 상품화 및 판매를 지원합니다.

ICE5 + Commerce Package + etc

서비스특징

  • 실물 상품 외에도 디지털 콘텐츠나 온라인 구독 서비스에도 다양한 형태의 커머스와 통계 기능 제공
  • 직관적인 WYSIWYG 방식의 상품 전시와 미리보기
  • 고객 개개인의 프로파일 데이터와 상황에 맞춘 개인화 추천 콘텐츠와 상품 노출

제공기능

단일 관리 체계에서의 멀티몰 지원

디지털 콘텐츠의 상품화 및 온라인 구독 서비스 판매 지원

단품, 패키지, 멤버십, 기간 결제, 정기결제 등 다양한 판매 옵션 제공

개인화 추천 컴포넌트, 인기 상품, 연관 상품 컴포넌트 등 전시 관련 컴포넌트 지원

Commerce Package

타겟팅 대상을 분류후, 원하는 마케팅 시나리오를 제작하여 자동으로 고객에게 콘텐츠를 발송하는 마케팅 오토메이션을 제공합니다.

통합 커머스 플랫폼 제공

B2C, B2B, B2E등 다양한 커머스서비스가 가능하도록 운용사, 채널(고객사), 판매자, 공급사 간 운영 플랫폼관리 기능을 제공합니다.

다양한 설정 기반의 관리 시스템 제공

채널기준 설정
  • 상품필터링 : 채널별 상품전시 기본 필터링
  • 상품 판매정보 : 상품 판매가, 재고 등 판매정보 채널별 설정
  • 메뉴/ 전시카테고리 : 채널별 메뉴/ 전시 카테고리 설정
  • 기획전 : 판매사샵 기획전을 기본으로 채널별 기획전 전시설정
  • 상품쿠폰 : 운용사 or 직영판매사가 채널별 상품쿠폰 설정
사이트기준 설정
  • 사이트 : 단일 채널 내 복수 사이트 동일 적용설정
  • 고객 : 사이트별로 고객 대상 적용
  • 브랜드사이트 : 판매사의 홍보/브랜딩을 위한 사이트 설정
  • 이벤트: 사이트별 이벤트 설정
  • 주문서 쿠폰 : 사이트별 주문서 쿠폰 설정. 발급주체는 운용사 or 직영판매사
사용자그룹 기준 설정
  • 사용자 그룹은 단일 고객사 or 멀티 고객사 모두 사용자 그룹설정이 가능하며, 특 정 그룹에 특화된 상품전시 설정 제공
  • 사용자그룹 특화상품 전시 : 특정사용자 그룹에게만 상품전시 설정 및 상품 판매정 보(판매가, 재고 등) 설정 제공

서비스 주요 기능

컴포넌트를 통한 전시관리

- 페이지 저작 동시에 화면 미리보기가 가능한 높은 직관성 제공 (페이지 저작과 전시 관리 통합)
- 컴포넌트에 API 바인딩 또는 아이템 직접 선택으로 데이터를 매핑하거나, 컴포넌트 개별 설정으로 개인화 노출 컨트롤 가능

템플릿/Rule 기반의 전시관리시스템

관리자 편의성을 위하여 컴포넌트 기반 UI 관리 방식을 적용하며 최신 웹 프레임웍 기술인 React.js를 활용하여 서비스페이지가 구현될 수 있도록 함

통합 상품관리기능 제공

B2B, B2C, B2E 사업모델을 수용하고, 상품정보를 활용한 다양한 서비스 구성을 지원하는 상품정보 관리체계를 구축함

컴포넌트 기반의 인터렉티브한 상품소개 구성기능 제공

단순구조의 스크롤 방식이 아닌, 컨텐츠빌더의 컴포넌트 기반의 풍부하고 인터렉티브한 상품소개 구성기능 제공

템플릿 기반 상품등록기능 제공

템플릿 기반 상품등록을 통한 필수정보 등록 및 카테고리별 상품 특성정보 반영과 자주 사용되는 정보 재사용 기능 제공
