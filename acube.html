대용량 파일 업로드

다양한 업로드 지원 및 분할 업로드

대용량 파일도 빠르게 업로드할 수 있도록 분할 업로드 기술을 사용하였습니다. 분할 업로드 기술로 인해, 업로드 작업이 시스템 운영과 독립적으로 동작하여 보다 안정적인 운영이 가능합니다.

태그(TAG) 검색

손쉬운 자료 식별, 분류, 검색을 위한 태그 추출

이미지 업로드 시, 이미지 정보 분석을 통한 자동 태그(TAG)를 추출합니다. 추출된 태그(TAG)를 기반으로 쉽고 빠른 검색이 가능합니다.

다양한 방식 업로드

서비스 이용에 영향이 없도록 분리 업로드 채택

파일 업로드 시 파일 개별 업로드, 폴더 전체 업로드, 대량 업로드 방식 모두 지원합니다. 파일 업로드 결과를 토스트 메시지창과 알림창을 통해 확인할 수 있습니다.

멀티 디바이스 대응

자동 룰셋에 의한 멀티 디바이스용 리소스 제작

이미지, 비디오, 오디오 유형별 상세 변환 설정 기능을 지원합니다. 서비스 목적에 따라 미디어 변환 설정을 통해 멀티 디바이스 콘텐츠를 생성할 수 있습니다.

인스트림 타겟 광고 제작

광고 유형에 맞춘 영상 편집

AssetCube를 통해 쉽고 빠른 광고 영상 제작을 할 수 있습니다. 광고 영상 제작시, 노출 시간 또는 SKIP 유무 등 다양한 옵션 설정이 가능합니다. (* Trueview Instream / * Bumper 광고)

  • * 인스트림 : 동영상 콘텐츠 내에 삽입되는 광고
  • * Trueview Instream : 동영상 광고를 시청한 지 특정 시간이 지나고 건너뛰기(SKIP)가 가능한 광고 형식
    • * Bumper : 건너뛰기 (SKIP)가 불가능한 광고 유형

마케팅용 광고영상 제작 (Clip Merge)

다양한 옵션을 통한 손 쉬운 영상 편집

주요 이벤트 영상 클립과 광고 영상을 병합하여 하이라이트 클립 영상을 제작할 수 있습니다. 필요에 따라, 영상 클립을 편집하여 서비스용 클립으로 재가공할 수 있습니다.

서비스 주요 기능

이미지 편집

- 이미지 콘텐츠에 대한 개별 편집 기능
- 전문 기술이 필요 없는 손쉬운 편집 가능
- Crop, Flip, Rotate, Draw, Shape, Icon, Text, Mask Filter 기능

동영상 편집

- 영상 콘텐츠에 대한 개별 편집 기능
- 별도 영상 편집 프로그램이 필요 없는 간단한 편집
- Rotate(회전), Speed(속도 조절), Text(글자 배치), Trim(구간 영상 추출), Merge(영상 합치기) 기능

자산 변환 룰셋

- 설정 기반으로 콘텐츠를 목적에 맞게 변환 가능
- 콘텐츠 타입에 적합한 변환 옵션 (확장자, DPI, 품질, 필터, 색상, 워터마크, 해상도 등)
- 목적에 따라 콘텐츠에 프리셋을 적용하여 서비스 가능

중앙 집중 관리

- 등록된 자산은 권한을 적용하여 다운로드, 북마크, 공유 등 기능
- 프리셋 기능을 활용하여 자산을 변환하거나 자산 특징과 맞는 컬렉션에 재분류
- 상세 패널에서 자산의 태그(TAG), 사용기간, 통계 등을 관리/확인

자동 TAG 추출

- 이미지 업로드 시, Google Vision API를 통해 자동으로 관련 태그(TAG) 생성
- 머신러닝 학습을 통해 이미지를 사전에 정의된 수백만 개의 카테고리로 빠르게 분류
- Google Vision API 사용 유무는 유료 설정 메뉴를 통해 손쉽게 설정

템플릿 제작

- 컬렉션에 등록된 자산을 템플릿에 매핑하여 마케팅용 디자인 템플릿을 제작 기능 지원
- WYSIWYG방식을 통해 손쉽게 자산을 매핑하여 마케팅 활용 목적에 맞게 서비스 가능
